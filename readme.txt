## Description

The project aims to predict the toxicity of molecular structures by converting them into graphs and leveraging Graph Neural Networks (GNNs). The data is preprocessed, and a GNN model is trained for toxicity prediction.

## Requirements

- Python (3.7 or later)
- PyTorch (1.7.1 or later)
- RDKit (2021.03.5 or later)
- scikit-learn (0.24.2 or later)
- torch-geometric (1.7.2 or later)
